# ros-stdserial
Serial node (ROS)
This is a package which contains a single node called comms.

# Arduino
upload the provided sketch on to an arduino with a dfrobot lcd sheild. 

# ROS
install:
place package in catkin_ws/src
-> catkin_make 
-> catkin_make install
run:
-> roscore
-> rosrun turtlesim turtlesim_node
-> rosrun piglet comms
